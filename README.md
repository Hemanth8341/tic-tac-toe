# Tic-Tac-Toe

A modern, responsive, and animated Tic-Tac-Toe game built with HTML, CSS (custom + Bootstrap 5), and JavaScript. Playable in any browser and ready for instant deployment to Netlify or any static hosting service.

## Features
- Classic 3x3 Tic-Tac-Toe gameplay
- Responsive, mobile-friendly design
- Animated winner highlighting and confetti celebration
- Clean, efficient codebase

## Folder Structure
```
importing/tic-tac-toe/
├── index.html
├── index.js
├── style.css
├── src/
│   └── tic-tac-toe.js
└── README.md
```

## How to Run Locally
1. Open `index.html` in your browser.
2. Play!

## How to Deploy to Netlify
1. Drag and drop the `importing/tic-tac-toe/` folder into Netlify, or connect your GitHub repo and set the publish directory to `importing/tic-tac-toe`.
2. No build command is needed.
3. Click "Deploy site".

Enjoy your game!
